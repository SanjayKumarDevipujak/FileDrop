CLI Commands

filedrop send <file>
Starts a room and prepares to send the specified file. Generates a room code and QR code.
filedrop receive <code>
Joins a room using the provided code to receive a file directly to your current directory.
filedrop --help
Shows all available commands and options.

Note: The CLI version currently uses a signaling server to establish P2P connections. For best results, ensure both devices have internet access.
